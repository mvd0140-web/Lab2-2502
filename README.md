### 1. Задание №1
>Посмотреть, какие значения принимает последовательность при разных `a0` и `n`, вывести закономерность, по которой строится последовательность и запрограммировать ее самостоятельно.

Я проанализировала последовательность и вывела закономерность, какие значения принимает переменная: каждый новый член последовательности получается умножением произведения цифр предыдущего члена на `7`.
#### Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class okeoke {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    static int pro(int a) {
        int q = a;
        int w = 1;
        if (q <= 0) {
            return 0;
        }
        else {
            while (q > 0) {
                int z = q % 10;
                w = w * z;
                q = q / 10;
            }
            return w;
        }
    }

    public static void main(String[] args) {
        int n = in.nextInt();
        int a1 = in.nextInt();
        int [] a = new int[n + 1];
        a[0] = a1;
        for (int i = 1; i < n + 1; i++) {
            a[i] = pro(a[i-1]) * 7;
        }
        for (int i = 1; i < n + 1; i++) {
            out.print(a[i] + " ");
        }
    }
}

```
#### Анализ правильности решения
Переменная `n` принимает значение длины последовательности.
Переменная `a1` принимает значение первого элемента последовательности.

### 2. Задание №2

>Дана последовательность из `n` чисел. Префиксом будем называть отрезок последовательности, который начинается с первого элемента и заканчивается на некоторой позиции. Найти первый префикс, на котором минимум и максимум имеют одинаковую четность, и вывести его длину (если такого префикса нет, вывести `0`). Первая строка: число `n`, вторая строка: `n` целых чисел – элементы последовательности.

так как нет смысла останавливаться на том, что первый префикс, у которого максимум и минимум будет одинаковым - это первый элемент последовательности, то я поставила условие, что длина отрезка должна быть больше `1`. Чтобы решить эту задачу, я последовательно ищу минимум и максимум последовательности и проверяю их на одинаковую четность. Если у них разная четность, ищу следующий минимум или максимум, затем проверяю. Если такого префикса нет - выводится `0`.

#### Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        int n = in.nextInt();
        int [] a = new int[n];
        for (int i = 0; i < n; i++) {
            a[i] = in.nextInt();
        }
        int min = a[0];
        int max = a[0];
        boolean f = false;
        for (int i = 1; i < n; i++) {
            if (a[i] < min) {
                min = a[i];
            }
            if (a[i] > max) {
                max = a[i];
            }
            if (min % 2 == 0 && max % 2 == 0 || min % 2 != 0 && max % 2 != 0) {
                out.println(i+1);
                f = true;
                break;

            }
        }
        if (!f) {
            out.println(0);
        }
    }
}
```

Затем вводится сама последовательность.

### 3. Задание №3

>Ввести с клавиатуры натуральное число `N`. Найти и вывести первые простые близнецы, которые больше числа N. Два простых числа называются близнецами, если разность между ними равна `2`.

Я создала подпрограмму, проверяющую число на простоту. Затем, я перебираю числа, большие `n`, если числа, разница между которыми `2`, являются простыми, вывожу их на экран и завершаю цикл.

#### Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    static boolean pr(int a) {
        if (a == 2 || a == 3) {
            return true;
        }
        for (int i = 2; i < a; i++) {
            if (a % i == 0) {
                return false;
            }
        }
        return true;
    }

    public static void main(String[] args) {
        int n = in.nextInt();
        int n1 = n + 1;
        while (true) {
            if (pr(n1) && pr((n1+2))) {
                out.println(n1 + " " + (n1 + 2));
                break;
            }
            n1++;
        }
    }
}

```

#### Анализ правильности решения
Переменная `n` принимает значение числа, после которого необходимо найти пару простых чисел.

### 4. Задание №4

>Дан массив из `n` чисел. Найти минимальную разницу между любыми двумя элементами массива. Первая строка: число `n`, вторая строка: `n` элементов массива.

Для того, чтобы найти минимальную разницу между любыми двумя элементами массива, необходимо присваивать в новую переменную значение разницы между двумя элементами, а затем эту переменную сравнивать со значением минимума (за значение минимума я взяла разницу между первым и вторым элементом). Если новая переменная оказалась меньше значения минимума, необходимо обновить значение минимума. В конце выводим значение минимума на экран. Это и будет наименьшая разница между двумя любыми элементами массива.

#### Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class okeoke {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        int n = in.nextInt();
        int [] a = new int[n];
        for (int i = 0; i < n; i++) {
            a[i] = in.nextInt();
        }
        int min = Math.abs(a[1] - a[0]);
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                int q = Math.abs(a[j] - a[i]);
                if (q < min) {
                    min = q;
                }
            }
        }
        out.println(min);
    }
}
```
